# ⏱️ Web Cronômetro

Um cronômetro web simples, responsivo e com visual moderno, desenvolvido com **HTML**, **CSS** e **JavaScript**.

## 📖 Sobre o projeto

O **Web Cronômetro** é uma aplicação front-end que permite iniciar, pausar e resetar a contagem do tempo diretamente no navegador.

O projeto foi desenvolvido com foco em praticar conceitos fundamentais do desenvolvimento web, como manipulação do DOM, controle de eventos, lógica de tempo com `setInterval` e responsividade para diferentes tamanhos de tela.

## ✨ Funcionalidades

- ▶️ Iniciar o cronômetro
- ⏸️ Pausar a contagem
- 🔄 Resetar o tempo
- 🕒 Exibição em **minutos**, **segundos** e **milissegundos**
- 📱 Layout responsivo para dispositivos móveis
- 🎨 Interface estilizada com tema escuro e fonte digital

## 🛠️ Tecnologias utilizadas

- **HTML5**
- **CSS3**
- **JavaScript**
- **Google Fonts (Orbitron)**

## 💻 Como funciona

O cronômetro utiliza JavaScript para controlar a contagem do tempo com `setInterval`, atualizando os elementos da interface em intervalos de 10 milissegundos.

As principais ações disponíveis são:

- **Iniciar:** começa a contagem do tempo
- **Pausar:** interrompe temporariamente o cronômetro
- **Resetar:** reinicia os valores para `00:00:000`
